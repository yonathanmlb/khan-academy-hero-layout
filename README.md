# Khan Academy Hero Layout

This project recreates a Khan Academy inspired homepage hero section using HTML and CSS. The goal of the project was to practice building modern landing page layouts and responsive design using Flexbox and layered visual elements.

## Preview

Open the project with Live Server or open `index.html` directly in your browser.

## Layout Reference

The layout was recreated from a design reference inspired by the Khan Academy homepage.

## Technologies Used

* HTML5
* CSS3
* Flexbox
* Responsive Design

## Features

* Responsive navigation bar
* Hero section with headline and call-to-action buttons
* Layered collage style image composition
* Modern landing page layout
* Mobile responsive design

## What I Practiced

* Building hero sections for landing pages
* Structuring layouts using Flexbox
* Positioning elements with CSS
* Responsive design for desktop and mobile
* Recreating real-world website interfaces
